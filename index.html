About this Project

COVIDMD is an attempt to track, aggregate, and display publically available COVID-19 data in a single, user friendly location

Data Sources

This website aggregates data from the following public sources:

https://coronavirus.maryland.gov/ https://raw.githubusercontent.com/nytimes/covid-19-data/master/us-counties.csv https://en.wikipedia.org/wiki/2020_coronavirus_pandemic_in_Maryland
